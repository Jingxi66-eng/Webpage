Flow scheme of quantitative analysis

  1. Starts from the sustainability issues
    which includes 3 aspects:
    Planet, People, Profit
  2. Implement system analysis (engineering table)
  3. Select indicators and perform a sensitivity analysis
  4. Compose different scenarios
  5. Perform a trade-off analysis
  6. Integrated assessment

Take away

  • How to analyze a system structurally
  • How to determine the function of a system
  • How to evaluate different scenarios for a system
  • The way to propose a prototype for an innovative system
