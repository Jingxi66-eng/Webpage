Structural design method

Reflexive Interactive Design (RID)

System Innovation

Main steps:

  • Indentify current problems in the system
  • Root cause analysis
  • Define key challenges
  • Determine system boundary
  • Clarify desgin objectives and challenges
  • Implement stakeholder analysis
  • Define system requirements
  • Find corresponding functions (IDEF0 diagram)
  • Implement function analysis and define key functions
  • Defien different scenarios
  • Scenario evaluation (weighted decision matrix)
  • Propose final designed system

Take away

  • How to analyze a system structurally
  • How to determine the function of a system
  • How to evaluate different scenarios for a system
  • The way to propose a prototype for an innovative system
